== Description ==
Blotout EdgeTag is a Personalization Platform for Growth Marketers. We enable the restoration of ID when cookies are deprecated. Our current offering allows advertisers to restore post-iOS identity issues and stitch users across devices when consented.

== Installation ==

1. Find the plugin in the list in WordPress Admin and click to install it. You can also upload the ZIP file through the admin backend.
2. Install the plugin
3. Go to [EdgeTag App](https://app.edgetag.io) and create an account.
4. Complete the onboarding process and get your EdgeTag URL.
5. Go back in WordPress and pate the EdgeTag URL in the plugin settings.
